To run the app, you can double-click on the project1.exe

many windows will apear, The main window in the center is to control 
the app (change color or quit)

you can show/hide all the options in each by double-click on it

then choose a time on the dropdown so that you can start the countdown

all the action are stored in the file "here.txt"

To quit the app, you can double-click on the last window and click on the option button 
then click on the "x" button on the controller window
